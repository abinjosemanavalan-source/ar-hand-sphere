# ar-hand-sphere
# 🖐️ AR Hand Studio – Gesture Controlled 3D Object Manipulation


## 📖 Overview

**AR Hand Studio** is a real-time Augmented Reality application that allows users to manipulate interactive 3D objects using only hand gestures captured by a webcam.

Using **MediaPipe** for hand tracking, **OpenCV** for image processing, and **PyOpenGL** for rendering, users can:

* Rotate 3D objects
* Scale objects using pinch gestures
* Switch between different 3D shapes
* Freeze object rotation
* Hide and reveal objects

No gloves, markers, or additional hardware are required.

---

# ✨ Features

✅ Real-time hand tracking

✅ 3D Cube, Sphere, and Pyramid

✅ Smooth gesture recognition

✅ Single-hand and dual-hand controls

✅ Transparent OpenGL rendering

✅ FPS counter

✅ Gesture detection HUD

✅ Automatic fallback to 2D mode if OpenGL is unavailable

# 🎮 Controls

| Gesture          | Action               |
| ---------------- | -------------------- |
| 👍 + ☝ + ✌       | Show & Rotate Object |
| 🤏 Thumb + Index | Scale Object         |
| ✌ Peace Sign     | Switch Shape         |
| ✋ Open Palm      | Freeze Rotation      |
| ✊ Fist           | Hide Object          |
| 🙌 Two Hands     | Scale Object         |

# ⚙ Installation

Clone the repository

```bash
git clone https://abinjosemanavalan//ar-hand-speher.git
```

Move into the project folder

```bash
cd AR-Hand-Studio
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python main.py
```
